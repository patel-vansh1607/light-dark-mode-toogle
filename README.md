## 📦 **Installation**
```bash
# Clone the repo
 git clone https://github.com/patel-vansh1607/light-dark-mode-toogle

# Navigate into the project
 cd light-dark-mode-toogle

# Install dependencies
 npm install

# Run the app
 npm start
```
